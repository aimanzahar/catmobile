## Background

Many cat grooming businesses still rely on traditional appointment methods — phone calls, walk-ins, and social media messaging — leading to scheduling conflicts, double bookings, and poor record-keeping. This application provides a dedicated digital booking platform to streamline grooming appointments for both pet owners and service providers.

## Problem Statement

Without a centralized digital system, cat grooming businesses face disorganized scheduling, limited real-time availability, and inconsistent customer communication. Pet owners struggle to track past visits, check available time slots, or receive timely appointment reminders. This app addresses these gaps with a structured, mobile-first booking solution.

## Objectives

1. Analyze user needs for a digital booking platform and assess shortcomings of traditional grooming appointment systems.
2. Design and implement a mobile-based cat grooming service reservation system.
3. Evaluate the system's usability and performance through testing.

## Getting Started

### Prerequisites

- PHP 8.1+
- Composer
- Node.js
- Appwrite instance (self-hosted or cloud)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd catfyp

# Install dependencies
composer install
npm install

# Configure environment
cp .env.example .env
# Update .env with your Appwrite credentials

# Run the application
php artisan serve
```
